Lehrgänge

Das Ausbildungsprogramm zum Web Professional ist modular aufgebaut und startet mit dem Basislehrgang zum Web Integrator. Dieser bildet die Grundlage für die beiden folgenden Diplomlehrgänge zum Web Designer und Web Developer. Danach hast du weitere Möglichkeiten, dich zu spezialisieren: Zum Beispiel zum Drupal oder Typo3-Entwickler, WordPress Spezialisten, mobile Developer oder Projektleiter. Gelernt wird Hand in Hand: Eine Crew von Spezialisten bringt dir alles bei, was du in der Praxis auch wirklich brauchst – und steht dir mit Rat und Tat bei deinen Web-Projekten zur Seite.

Die Ausbildung wird ergänzt durch die halbjährlich stattfindenden Web Professionals Camps, die Web Professionals and Friends Treffen (alle zwei Monate), regelmässige Kurse und Workshops, sowie die Recruiting Days.
